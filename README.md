    `–ü—Ä–∏—è—Ç–Ω–æ –ø–æ–∑–Ω–∞–∫–æ–º–∏—Ç—å—Å—è, ${state.data.name}! –°–∫–æ–ª—å–∫–æ –≤–∞–º –ª–µ—Ç?`
